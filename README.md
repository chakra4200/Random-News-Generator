Random News Generator

Description

This project is a console-based Python application that generates random news headlines using Python's built-in "random" module. It demonstrates the use of lists, loops, conditional statements, and random selection to create different news headline combinations.

Technologies Used

- Python
- Random Module

Features

- Generates random news headlines
- Uses lists to store headline components
- Demonstrates random selection
- Menu-driven console application
- Beginner-friendly Python logic

Concepts Used

- Lists
- Loops
- Functions
- Conditional Statements
- Random Module

How to Run

1. Install Python.
2. Open the project folder.
3. Run the Python file.
4. Follow the on-screen instructions to generate random headlines.
